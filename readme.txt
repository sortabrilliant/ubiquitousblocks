Let's make reusable blocks even easier to reuse.

== Description ==

Reusable blocks are great but we want to make them even better. What if you could add a reusable block to every post/page with a few clicks of the mouse. You can with Ubiquitous Blocks.

= Where can I learn more? =

Learn more at [sorta brilliant](https://sortabrilliant.com/ubiquitous/).
